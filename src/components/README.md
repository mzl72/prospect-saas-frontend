**cadence/WeekCalendar.tsx**: Seletor visual de dias da semana + janelas de tempo
**cadence/cadence-utils.ts**: Utilitários (cores, labels, formatação) para componentes de cadência
**ui/**: Componentes Shadcn/UI (badge, button, card, dialog, form, input, label, progress, select, slider, switch, toaster)
