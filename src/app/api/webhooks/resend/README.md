# resend - Webhook Resend (Tracking de Emails)

**route.ts**: Webhook POST que recebe eventos de emails do Resend. Validação: HMAC signature via validateResendWebhookSignature (headers 'svix-signature' ou 'webhook-signature' contra RESEND_WEBHOOK_SECRET). Eventos processados: 'email.sent' (aceito pelo Resend, apenas log), 'email.delivered' (entregue ao servidor destinatário, apenas log), 'email.opened' (lead abriu email, chama handleEmailOpened: busca Email por resendId, atualiza openedAt, incrementa openCount), 'email.clicked' (lead clicou em link, chama handleEmailClicked: registra clickedAt), 'email.bounced' (email bounced, chama handleEmailBounced: atualiza status BOUNCED + bouncedAt, atualiza Lead.status para BOUNCED), 'email.complained' (marcado como spam, apenas log). Interface: ResendWebhookEvent com type, created_at, data (email_id/resendId, from, to[], subject, link para clicks).
