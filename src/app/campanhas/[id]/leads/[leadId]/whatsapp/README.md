# whatsapp - Histórico WhatsApp do Lead

**page.tsx**: Página dedicada ao histórico completo de mensagens WhatsApp de um lead. Breadcrumb de navegação (campanha → lead → whatsapp). Exibe todas mensagens WhatsApp ordenadas por sequenceNumber (1, 2, 3...), com phoneNumber, message completa, status com cores mapeadas (PENDING: gray, SENT: blue, DELIVERED: cyan, READ: green, FAILED: red), timestamps sentAt/deliveredAt/readAt formatados. Interface Lead inclui dados básicos + análise IA (companyResearch, strategicAnalysis, personalization, analysisLink).
